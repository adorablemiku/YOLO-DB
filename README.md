# YOLO-DB
Deformable Convolutions and Bidirectional Weighted Fusion for Enhanced Small-Pest Detection in Agricultural Imagery
# Experimental Details and Environment
The experiments were conducted on a computing platform equipped with an Intel(R) Gold 6258R CPU running at 2.70 GHz, 256 GB of RAM, and an A100 Tesla GPU with 40 GB of video memory, under an Ubuntu 18.06 64-bit operating system. All models were trained using Python 3.7, PyTorch 1.12.1, CUDA 11.4, and cuDNN 8.4.1.
